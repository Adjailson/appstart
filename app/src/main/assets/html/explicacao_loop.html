O comando input espera o usuário digitar algo para ser atribuído a variável nota1, assim também acontece em nota2.

Em seguida a variável media recebe a soma das notas divididas por 2.
Depois precisamos fazer uma verificação de parada do while usando if e elif.

Se if for verdadeiro retornamos uma mensagem e para o while com o uso do comando break.
Se elif for verdadeiro retornamos um mensagem e o programa volta para a primeira linha e pede para digitar valores novamente nas variáveis nota1 e nota2.

O que tem a frente do símbolo # não é executado pelo código, é usado em Python para comentar algo.
